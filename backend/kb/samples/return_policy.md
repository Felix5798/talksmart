# 退货政策

本政策适用于本平台自营商品。第三方商家商品以商品页及商家公示规则为准。

## 适用范围

- 自营商品：签收后符合下列条件的，可申请退货退款。
- 虚拟商品、定制类商品、生鲜易腐类商品不适用无理由退货（详见「不可退情形」）。

## 退货时限

- 签收后 **7 个自然日内** 可申请无理由退货。
- 商品须 **未使用、未洗涤**，吊牌、防伪标识、配件、赠品、包装完好。
- 超过 7 天但仍在 **15 天内**，仅支持因 **质量问题** 申请退货。

## 不可退情形

以下情形不支持退货或需人工审核：

- 贴身衣物、内衣裤、袜子等一经拆封即影响二次销售的商品。
- 定制、刻字、个性化加工商品。
- 已激活的数码产品（手机、平板等），除非存在质量问题。
- 生鲜、冷冻、保质期敏感食品。
- 用户自行损坏、污染、改装的商品。

## 退货流程

1. 在「我的订单」中选择对应订单，点击「申请退货」并填写原因、上传凭证（质量问题必填）。
2. 客服或系统审核，一般 **1–2 个工作日** 内反馈。
3. 审核通过后，按页面指引寄回商品；请勿使用到付。
4. 仓库验收合格后发起退款；验收异常将联系用户说明。

## 运费承担

- **质量问题** 或 **发错货、少件**：运费由平台承担，可提供运费凭证后报销。
- **无理由退货**：退货运费由 **买家承担**。
- 未经审核自行寄回导致无法识别的包裹，平台有权拒收或延长处理时间。

## 退款方式与时效

- 退款原路退回至下单支付账户（微信、支付宝、银行卡等）。
- 仓库确认收货后 **3–15 个工作日** 到账，具体以支付渠道为准。
- 使用优惠券、满减的订单，退款金额按实付比例折算，券规则以活动说明为准。

## 换货说明

- 支持 **同款同色同尺码** 换货，需先申请退货再重新下单，或联系在线客服办理换货（库存允许时）。
- 换货不额外收取商品差价；运费规则与退货一致。

## 特殊说明

- 大促期间（如 618、双 11）订单量较大，审核与退款可能延长 **1–3 个工作日**。
- 跨境商品退货需遵守海关相关规定，已清关入境商品可能无法退回海外仓。
- 组合套餐、满赠活动商品退货时，未退回的赠品将按赠品价值在退款中扣除。

## 联系与升级

- 在线客服工作时间：每日 9:00–22:00。
- 对处理结果有异议，可在订单详情申请 **人工复核** 或拨打客服热线 400-xxx-xxxx。
